📱 Configuración WhatsApp

Los datos se enviarán directamente a tu WhatsApp como mensaje estructurado.

Ejemplo de mensaje que recibirás: 🆕 Nueva solicitud de envío - Uñas de la Monse 👤 Cliente: María García 📱 Teléfono: 0987654321 🆔 Cédula: 1234567890 📍 Dirección: Av. Principal #123, Quito 🛍️ Producto: Esmalte rojo mate, cantidad: 2 📸 Foto adjunta en el chat

📧 Configuración Email

Usa un servicio como EmailJS para recibir emails automáticamente sin servidor.

Para configurar EmailJS: 1. Registrate en emailjs.com (GRATIS) 2. Crea un servicio de email 3. Configura tu plantilla 4. Obtén tu Service ID y Template ID 5. Pégalos en el código JavaScript

📝 Configuración Google Forms

Conecta con Google Forms para almacenar automáticamente en Google Sheets.

Pasos para configurar: 1. Crea un Google Form con los mismos campos 2. Ve a "Respuestas" → "Crear hoja de cálculo" 3. Copia la URL del formulario 4. Los datos se guardarán automáticamente en Sheets

💬 Configuración Telegram

Recibe notificaciones instantáneas en tu Telegram Bot.

Para crear tu bot: 1. Habla con @BotFather en Telegram 2. Usa /newbot y sigue las instrucciones 3. Obtén tu token del bot 4. Usa @userinfobot para conocer tu Chat ID
